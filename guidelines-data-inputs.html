Data Inputs Guide

Panduan ini menjelaskan cara mengisi formulir data harian untuk masing-masing site produksi. Pastikan semua field diisi dengan benar sebelum submit untuk menjaga keakuratan data di dashboard.

01
Akses Form Data Inputs

Buka halaman Data Inputs dan pilih site yang sesuai. Setiap site memiliki form Google Sheet tersendiri yang dapat diakses melalui tombol "Open Data Input App".

02
Field Wajib Diisi

Setiap form memiliki field wajib (required) yang harus diisi sebelum submit. Field wajib mencakup: tanggal, shift, nama operator, dan data produksi aktual (WMT/BCM/Kadar).

03
Format Data & Satuan

Gunakan format angka desimal dengan titik (bukan koma) untuk data numerik. Satuan standar: WMT untuk tonase, BCM untuk volume, %, Ni, Fe, Co untuk kadar mineral.

04
Data Produksi Harian

Input produksi harian mencakup: Mining (OB removal + ore production), Hauling (ritase & jarak), Stockpile (in/out), dan Shipment (loading & manifest) sesuai aktivitas di lapangan.

05
Data Peralatan & SDM

Isi data availability alat berat (PA/UA/MA), konsumsi BBM per unit, jam kerja efektif, data cuaca (curah hujan), dan jumlah manpower harian per shift.

06
Verifikasi & Submit

Setelah mengisi semua field, periksa kembali data sebelum menyimpan. Data yang telah disubmit akan otomatis terupdate di Dashboard dalam beberapa menit melalui Google Data Studio.
